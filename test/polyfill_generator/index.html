Tutorial:

You specify which polyfills you desire, such as "CE" or "WC-WA-_pe-PR". Polyfill short codes to make them usable and recognizeable across links / code and server / client.

  1. Dash: separate polyfill code
  2. Lowercase: polyfill loaded ASYNC
  3. Uppercase: polyfill loaded SYNC
  4. Underscore infront: does not halt "polyfill.ready()". This only applies when "polyfill.ready" framework is active.

If you want to specify your own time to run `polyfill.ready`, just add your own timer as described in chapter 8.
